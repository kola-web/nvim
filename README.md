# 插件使用备忘录

## abolish.vim

| 按键 | 作用                 |
| ---- | -------------------- |
| crm  | coerce to MixedCase  |
| crc  | coerce to camelCase  |
| cru  | coerce to UPPER_CASE |
| cr.  | coerce to dot.case   |
| cr-  | coerce to dash-case  |
| cr   | coerce to space case |
| crt  | coerce to Title Case |
